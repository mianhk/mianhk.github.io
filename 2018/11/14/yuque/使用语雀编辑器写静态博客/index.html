记录一些坑

  • 知识库一定要是公开的

  • windows 下更新 nodejs 不能用 n，即使强制使用了也还是无法更新

  • 新版本的 npm。如果改了 package.json，且 package.json 和 lock 文件不同，那么执行npm i时 npm 会根据 package 中的版本号以及语义含义去下载最新的包，并更新至 lock。
