服务器-Nginx模块化结构

习惯上将Nginx分为：核心模块、标准模块、可选HTTP模块、邮件服务模块和第三方模块五大类。

核心模块

包含对两部分功能的支持：

  • 主体功能：进程管理、权限控制、错误日志记录、配置解析等
  • 用于响应请求必需的功能：事件驱动机制、正则表达式解析等
